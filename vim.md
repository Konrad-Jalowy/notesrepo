# Vim notes
## Commands
- **show help:**
```sh
:help
```
- **quit - vim, help etc:**
```sh
:q
```
- **ignore changes and quit:**
```sh
:q!
```
- **execute pwd command:**
```sh
:pwd
```
- **change working directory:**
```sh
:cd C:\vimfiles
```
- **save unsaved file in current directory:**
```sh
:w nowy.txt
```
- **save and quit:**
```sh
:wq
```
