## NPM Commands
- **List installed packages:**
```sh
npm ls
```
- **Install package:**
```sh
npm install cowsay
```
- **Install global package:**
```sh
npm install -g cowsay
```
- **Uninstall package:**
```sh
npm uninstall cowsay
```
- **Create package:**
```sh
npm init
```
- **Create package no questions asked:**
```sh
npm init -y
```
