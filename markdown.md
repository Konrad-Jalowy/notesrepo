## Markdown format notes
**bold text**:
```md
**bold text here**
```
**bold italics text**:
```md
***bold italics here***
```
**crossed out text**
```md
~crossed out text here~
```
**headers from 1 to 6**
```md
# Space after hash always
```
**embedding code**
```md
```sh
echo "Remember about newlines before and after your code + theres no slash"
\```
```
